# Contributing

Thanks for your interest in contributing to **AnEshaEdit**.

## Quick start
- Fork the repo
- Create a branch from `main`
- Keep changes focused and small
- Open a PR with screenshots for UI changes

## Content updates (blog posts)
- Add posts in `_posts/` as Markdown.
- Use the `layout: post` frontmatter.
- Keep hero images optimized (prefer WebP when possible).

## Local preview
If you have Ruby + Bundler installed:

```bash
bundle install
bundle exec jekyll serve
```

Then open: http://localhost:4000/AnEshaEdit/
